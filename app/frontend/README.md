# 前端 App（smart_cane）

基于 uni-app（Vue）的跨平台手机应用，提供导航界面、BLE 设备连接、语音助手交互。

## 环境要求

- HBuilderX

## 运行

**HBuilderX 方式：**
1. 用 HBuilderX 打开
2. 菜单 → 运行 → 运行到浏览器 / 运行到手机

## 页面结构

| 页面 | 路径 | 功能 |
|------|------|------|
| 首页 | `pages/home/` | 主入口 |
| 地图 | `pages/map/` | 百度地图展示与导航 |
| 导航 | `pages/navigation/` | 步行导航模式 |
| 设备 | `pages/device/` | BLE 盲文设备连接管理 |
| 诊断 | `pages/diagnostic/` | 设备调试与状态检测 |
